Event Overview
Contestants will engage in a rhythm game, which is based off of a hit game: Friday Night Funkin'. Characters and various other sprites are based on an animated YouTube series called Battle for Dream Island (BFDI). This game is only accessible on iOS and can be played for free. The game requires performing a sequence of consecutive taps synced to music -- accurate and perfectly-timed taps will result in higher scores. Scores can be submitted as many times as needed; only the highest score will be saved for each level. Total accumulative scores from all tracks will be accounted for when ranking players in the event.

At the end of the event, top 5 players will be given rewards as follows:
1st — $15 iTunes + Free hyperPad Access Code + Discord Role
2nd — $10 iTunes + Free hyperPad Access Code + Discord Role
3rd — $10 iTunes + Free hyperPad Access Code + Discord Role
4th — $5 iTunes + Free hyperPad Access Code + Discord Role
5th — $5 iTunes + Discord Role
hyperPad Access Codes will allow you to purchase hyperPad for free!

Contestants must have Discord and reside in the official hyperPad Discord server to be able to participate and claim prizes. The game can be played using hyperPad OR the hyperPad Viewer app. For all of your scores to be valid, you must sign in using your discord on the game itself.

Event ends on May 15th, 2021 at 11:59PM EST. (FINALIZED DATE!)
