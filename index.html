hyperSpeed Event

Updated Jan 27, 2021

Event Overview
Contestants will speedrun multiple courses for the event -- scores will be recorded from valid runs and will be displayed in leaderboards. Having a shorter time (shorter run) and collecting more coins will result in higher scores. Scores can be submitted as many times as needed; only the highest score will be saved for each course. Total accumulative scores from all courses will be accounted for when ranking players in the event.

At the end of the event, top 3 players will be given rewards as follows:
1st — $10 iTunes + Free hyperPad Access Code + Discord Role
2nd — $5 iTunes + Free hyperPad Access Code + Discord Role
3rd — Free hyperPad Access Code + Discord Role
hyperPad Access Codes will allow you to purchase hyperPad for free!

Contestants must have Discord and reside in the official hyperPad Discord server to be able to participate and claim prizes. For all of your runs to be valid, you must provide your discord tag on the speedrun game itself -- check for spelling as providing an incorrect tag will deem your runs as invalid.

Event ends on February 27th, 2021 at 11:59PM EST.

Prohibited Behavior
The following behavior will disqualify contestants.

1. Modifying content in terms of hacks, mods and cheats to gain an unfair advantage over other contestants.
2. Creating multiple accounts to increase the overall likelihood of winning -- one account per user.
3. Failing to comply with rules in the Discord Server and the TOS of Discord.
